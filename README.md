# AI Chatbot com n8n

Projeto de estudo criando um chatbot usando:

- n8n
- Inteligência Artificial
- Google Sheets

## O que o bot faz

- recebe mensagens via webhook
- envia para um modelo de IA
- retorna uma resposta automática
- salva as conversas em uma planilha

## Estrutura

Webhook → AI Agent → Google Sheets
